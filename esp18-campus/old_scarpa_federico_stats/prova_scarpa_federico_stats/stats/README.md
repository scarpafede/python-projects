## Descrizione

Il programma crea una tabella in un database sqlite con dati provenienti dai primi due fogli dei file excel che si trovano nella cartella flussi.

Successivamente rinomina ogni file utilizzato aggiungendo l'estensione .bak .

## Requisiti

- Requisiti minimi di sistema;
- Python 3.10.7 (o superiore) installato;
- File specificati/o nella descrizione;
- Struttura "deploy" corretta;
- (Opzionale) un qualsiasi IDE che supporti Python.

## Esecuzione

Per eseguire il programma e' possibile utilizzare la linea di comando.

Esempio per Windows:
```
C:\scarpa_federico_stats\stats\bin>C:\Python310\python.exe stats.py
```

Dove "C:\scarpa_federico_stats\stats\bin" e' il percorso del programma, "C:\Python310\python.exe" e' il percorso dell'interprete Python e "stats.py" e' il nome del programma.

Esempio per Linux:
```
$ python3 stats.py
```
