## Descrizione

Il programma è un insieme di esercizi che vengono utilizzati per il recupero della materia Informatica.
È composto da due esercizi:

Il primo "str .py" lavora sulle stringhe e propone le seguenti funzionalità:

- contare le  occorrenze di un carattere in una stringa (l'implementazione non deve usare il metodo count() )
- stampare una stringa alla rovescia
- verificare se una stringa è palindroma (cioè si legge allo stesso modo da sinistra a destra o da destra a sinistra)

Il secondo "numeri .py" propone funzionalità in campo numerico:

- calcolo del M.C.D. (massimo comune divisore) di due numeri interi
- calcolo del valore di PiGreco come somma di n termini della serie:
-- se n = 6 π = 4/1 - 4/3 + 4/5 - 4/7 + 4/9 - 4/11
-- se n = 8 π = 4/1 - 4/3 + 4/5 - 4/7 + 4/9 - 4/11 + 4/13 - 4/15)

## Requisiti

Python 3.10.7
Windows 10/11
Almeno 4 gb di RAM
SSD/HDD da almeno 10 GB
CPU Intel core i3-5005U o superiore

## Esecuzione

Scaricare il/i file desiderati, aprire la IDLE di python, premere File in alto a destra, Open, selezionare il percorso dove si sono salvati i file scaricati, premere apri;
per eseguire il programma dalla finestra appena aperta, eseguirlo con F5 o con run nella barra dei menù in alto, se il programma chiede di salvare il progetto, fare clic su salva.
