## Descrizione

La consegna è composta di 2 file:

Il primo (my_log) scrive sul file execute.log il tempo che ha impiegato ad eseguirsi, tramite la sua versione user-friendly e la sua versione seriale (timestamp), insieme al nome del programma;

Il secondo (filter), legge il contenuto del file execute.log e lo scrive nel file filter.txt, filtrandolo nel modo che le stringhe contenenti "Inizio" nel log, vangano riportate nel secondo.

## Requisiti

- Python 3.10.7
- Windows 10/11
- Almeno 4 gb di RAM
- SSD/HDD da almeno 10 GB
- CPU Intel core i3-5005U o superiore

## Esecuzione

Scaricare il/i file desiderati, aprire la IDLE di python, premere File in alto a destra, Open, selezionare il percorso dove si sono salvati i file scaricati, premere apri;

per eseguire il programma dalla finestra appena aperta, eseguirlo con F5 o con run nella barra dei menù in alto, se il programma chiede di salvare il progetto, fare clic su salva.
