## Descrizione

Il programma calcola_bioritmi.py prende dal file alunni_4aa.csv il nome, il cognome e la data di nascita per ogni riga e riscrive su bioritmi_classe.csv il nome e il cognome, insieme alla data di esecuzione del programma e i bioritmi, calcolati dal programma tramite la data di nascita di ciascun alunno.

I file citati sopra sono presenti nella cartella flussi all'interno del deploy.

## Requisiti

- Requisiti minimi di sistema;
- Python 3.10.7 o un qualsiasi IDE che supporti python;
- File specificati/o nella descrizione;
- Struttura "deploy" corretta.

## Esecuzione

Per eseguire il programma è possibile utilizzare un qualsiasi IDE che supporti python oppure la linea di comando.

Un esempio di esecuzione tramite cmd di Windows:

    C:\scarpa_federico_bioritmi_classe\bioritmi_classe\bin>C:\Python310\python.exe calcola_bioritmi.py

Dove "C:\scarpa_federico_bioritmi_classe\bioritmi_classe\bin" è il percorso della cartella contenente il file python da eseguire, "C:\Python310\python.exe" è il percorso dell'interprete e "calcola_bioritmi.py" è il nome del programma.
