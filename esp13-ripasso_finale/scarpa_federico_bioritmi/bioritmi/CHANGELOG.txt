# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0] - 2023-12-04

### Released

## [2.0] - 2023-12-19

### Added

- Supporto per CLI con input.
- Funzione "gestione_input", fa terminare il programma al verificarsi
  di condizioni specifiche descritte in essa.

### Changed

- Il nome della funzione di log è stato cambiato da "logmessage" a "writelog".
- Il programma ora calcola i bioritmi a partire da una data e un numero
  di giorni successivi, entrambi forniti tramite linea comando.

## [3.0] - 2024-01-08

### Added

- Lettura dal file csv e creazione di un html con una tabella contenente questi.

## [4.0] - 2024-01-18

### Added

- File batch che esegue il programma in modo automatico e mostra i risultati.
- Colori nella tabella html per evidenziare il segno dei bioritmi.

### Changed

- Ora la funzione "passaggio_html" viene chiamata nel main.
- Miglioramenti sintattici/lessicali nel file.
